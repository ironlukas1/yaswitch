# Shortcut Quick Reference

See full integration snippets in `docs/integration/shortcuts.md`.

Canonical apply command:

```bash
yaswitch apply --theme ~/.config/yaswitch/themes/minimal --compositor sway --json
```

Cycling command:

```bash
yaswitch cycle --compositor sway --json
```

Install generated shortcut include file:

```bash
yaswitch install-shortcut --compositor sway
```

This workflow is CLI-only (no visible GUI window).
